# Jaaveed Portfolio

Personal developer portfolio built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, and Three.js.

## Local Development

```sh
npm install
npm run dev
```

## Quality Checks

```sh
npm run lint
npm test
npm run build
```

## Deployment

The app builds to `dist` and is configured for GitHub Pages through GitHub Actions and for Netlify through `netlify.toml`.

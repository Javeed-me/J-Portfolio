#  J Portfolio

A modern, responsive developer portfolio showcasing projects, skills, and experience with smooth animations and clean UI.

---

## Overview

This portfolio is designed to present my work, technical skills, and projects in an interactive and visually appealing way. It includes smooth transitions, dynamic routing, and a clean user experience.

---

## Features

*  Modern UI with clean design
*  Smooth animations using Framer Motion
*  Multi-page navigation (Projects, Skills, Resume, Contact)
*  Fully responsive (mobile + desktop)
*  Gradient backgrounds and interactive elements
*  Notification system (Toaster / Sonner)
*  Optimized for performance and deployment

---

## Pages

* **Home** – Introduction and overview
* **Projects** – Showcase of my work
* **Skills** – Technical skills and tools
* **Resume** – Professional experience
* **Contact** – Get in touch

---

## Tech Stack

**Frontend**

* React
* Vite
* TypeScript

**UI & Styling**

* Tailwind CSS
* ShadCN UI

**Libraries**

* Framer Motion (animations)
* React Router (navigation)
* TanStack Query (state/data handling)
